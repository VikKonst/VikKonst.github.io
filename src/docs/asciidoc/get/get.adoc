= Получение задачи (GET)

Метод получения задачи.

== Входные параметры

Входные параметры приведены в таблице ниже.

.Входные параметры
|===
|*Параметр* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Идентификатор задачи
|task_id
|Число
|true
|Идентификатор задачи
|===

== Выходные параметры

Выходные параметры:

* в случае успеха вернуть параметры задачи, указанные в таблице;

* иначе вернуть описание ошибки.

.Параметры задач
|===
|*Параметр* |*Ключ* |*Тип* |*Не пустой* |*Описание*

|Заголовок задачи
|title
|Однострочный текст
|true
|Краткое описание задачи.

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать.

|Статус
|status
|Однострочный текст
|true
|Статус исполнения задачи.
|===
